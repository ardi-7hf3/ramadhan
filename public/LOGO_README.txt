# LOGO INSTRUCTIONS / INSTRUKSI LOGO

Letakkan file logo di folder `public/`:

```
public/
  mpk.png    ← Logo MPK
  osis.png   ← Logo OSIS
  favicon.svg
```

Logo akan otomatis tampil di kartu ucapan.
Jika file tidak ditemukan, akan tampil ikon SVG sebagai fallback.

---

Place logo files in the `public/` folder:
- `public/mpk.png`  → MPK logo
- `public/osis.png` → OSIS logo

If missing, SVG icon fallbacks will display automatically.
